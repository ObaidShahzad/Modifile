# Modifile
A tool to search and list recent modified files

# Requirements
+ Python 3.x
+ Linux Shell

# Usage

### Linux Shell
```sh
 ./modifile.sh [DIRECTORY] [DAYS]
``` 
#### Example:
```sh
./modifile.sh /home/ 2 
``` 
 
### Python 3
 ```sh
python3 modifile.py [DIRECTORY] [DAYS]
```
#### Example:
```sh
python3 modifile.py /home/ 2
```
